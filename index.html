如何取得 GitHub Token？
  1. 打開 GitHub，點右上角頭像。
  2. 點 Settings。
  3. 左側滑到最下面，點 Developer settings。
  4. 點 Personal access tokens。
  5. 選 Fine-grained tokens。
  6. 點 Generate new token。
  7. Repository access 選 Only select repositories，選 BYwayJu / gtav-map。
  8. Repository permissions 找到 Contents，設成 Read and write。
  9. 按 Generate token 後，複製 github_pat_ 開頭的 Token，貼到上方欄位。
Token 只會顯示一次，請不要公開給陌生人。平常看標記不需要 Token；只有按「上傳同步」才需要。
