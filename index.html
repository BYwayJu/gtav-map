如何取得 GitHub Token？
  1. 打開 GitHub，點右上角頭像。
  2. 進入 Settings。
  3. 左側滑到最下面，點 Developer settings。
  4. 點 Personal access tokens。
  5. 選 Fine-grained tokens。
  6. 點 Generate new token。
  7. Repository access 選 Only select repositories，選這個專案 gtav-map。
  8. Repository permissions 裡把 Contents 設成 Read and write。
  9. 產生後複製 github_pat_ 開頭的 Token，貼到上面的欄位。
Token 只會顯示一次。不要公開給陌生人；每個人只有要新增、移動、修改、刪除標記時才需要自己的 Token。
