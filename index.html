如何取得 GitHub Token？
  1. GitHub 頭像 → Settings。
  2. Developer settings → Personal access tokens。
  3. Fine-grained tokens → Generate new token。
  4. Repository access 選 Only select repositories，選 gtav-map。
  5. Repository permissions 選 Contents：Read and write。
