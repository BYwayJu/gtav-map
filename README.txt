GTAV 私人地圖標記 - GitHub 版

檔案說明：
- index.html
- markers.json
- map-atlas.png
- map-satellite.jpg
- map-roadmap.png

上傳方式：
1. 把整包檔案上傳到你的 GitHub repo 根目錄
2. 進入 GitHub Repo → Settings → Pages
3. Source 選「Deploy from a branch」
4. Branch 選 main / root
5. 儲存後等一下，網站就會出現

功能：
- 切換 3 張地圖（Atlas / Satellite / Roadmap）
- 手機雙指縮放、單指拖曳
- 電腦滑鼠滾輪縮放、拖曳
- 新增 / 命名 / 備註 / 刪除標記
- 多種圖示分類
- 可存在瀏覽器本機
- 可填 GitHub Token 同步 markers.json
